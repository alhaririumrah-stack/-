# نظام تشغيل العمرة ERP - نسخة موسعة كاملة

هذه نسخة موسعة تعمل على:
- GitHub Pages للواجهة
- Google Apps Script للـ API
- Google Sheets كقاعدة بيانات
- Google Drive للملفات عبر روابط Drive داخل ملف المعتمر

## الملفات
- index.html: صفحة الدخول
- admin.html: لوحة الإدارة
- portal.html: بوابة المعتمر
- config.js: رابط Apps Script فقط
- assets/app.js: منطق الواجهة
- assets/style.css: التصميم
- Code.gs: كود Google Apps Script
- .nojekyll: لمنع مشاكل Jekyll في GitHub Pages

## التشغيل
1. أنشئ Google Sheet فارغ.
2. افتح Extensions > Apps Script.
3. انسخ محتوى Code.gs.
4. اختر دالة setup ثم Run ووافق على الصلاحيات.
5. Deploy > New deployment > Web app.
6. Execute as: Me.
7. Who has access: Anyone.
8. انسخ رابط /exec.
9. ضعه في config.js مكان PUT_YOUR_APPS_SCRIPT_WEB_APP_EXEC_URL_HERE.
10. ارفع الملفات إلى GitHub بحيث يكون assets مجلد حقيقي في الجذر.

## ملاحظات قاسية ومهمة
- هذا نظام تشغيلي كامل قدر الإمكان فوق Google Sheets، لكنه ليس بديلاً قانونياً نهائياً عن برنامج محاسبة معتمد إذا توسعت الشركة كثيراً.
- لا ترفع ملف ZIP إلى GitHub Pages. ارفع الملفات مفكوكة.
- لا تضع app.js و style.css في الجذر. يجب أن يكونا داخل assets.
- بعد كل تعديل في Code.gs يجب نشر New version وأخذ رابط /exec الجديد إذا تغيّر النشر.
