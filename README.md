# نظام تشغيل العمرة ERP - Google Sheets + Apps Script + GitHub Pages

هذه نسخة موسعة جاهزة للتشغيل لشركة عمرة واحدة. ليست مجرد CRM، بل نظام عمليات مبني حول الرحلة كوحدة أعمال.

## الملفات
- `index.html`: تسجيل الدخول
- `admin.html`: لوحة الإدارة
- `portal.html`: بوابة المعتمر
- `assets/app.js`: منطق الواجهة
- `assets/style.css`: تصميم عربي RTL
- `config.js`: ضع فيه رابط Google Apps Script Web App
- `Code.gs`: Backend كامل لجوجل Apps Script

## التشغيل
1. أنشئ Google Sheet جديد.
2. من Extensions > Apps Script الصق محتوى `Code.gs`.
3. شغل دالة `setup` مرة واحدة وامنح الصلاحيات.
4. Deploy > New deployment > Web app.
5. Execute as: Me. Who has access: Anyone.
6. انسخ رابط Web App وضعه في `config.js` مكان `PASTE_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE`.
7. ارفع الملفات إلى GitHub Pages.
8. دخول المدير الافتراضي: `admin` / `123456`.

## الجداول التي ينشئها النظام
Users, Trips, Pilgrims, Payments, Expenses, Visas, Hotels, Rooms, Buses, BusStops, Guides, Notifications, Requirements, Attendance, Complaints, Logs

## الوظائف المضمنة
- إنشاء رحلات برية/طيران.
- تسجيل فرد أو عائلة من نفس الشاشة.
- ملف معتمر مع جواز، تأشيرة، رابط Drive، باقة، نوع غرفة، رصيد مالي.
- إنشاء حساب تلقائي للمعتمر.
- دفعات متعددة العملات وتحويلها إلى USD.
- مصاريف رحلة بتصنيفات.
- تأشيرات وتحديث حالة التأشيرة في ملف المعتمر.
- فنادق وغرف وسعة.
- باصات وسائقين ومشرفين.
- توقف باص واستراحة وعدد من نزل/عاد.
- مرشدون وعهد مالية.
- متطلبات رحلة تحسب حسب عدد المعتمرين.
- حضور وغياب.
- إشعارات تظهر في بوابة المعتمر.
- شكاوى وحالات طارئة.
- توزيع ذكي أولي للغرف والباصات.
- بطاقة معتمر قابلة للطباعة.

## ملاحظة قاسية ومهمة
Google Sheets مناسب كبداية قوية ومجانية، لكنه ليس منصة ERP ضخمة لآلاف العمليات المتزامنة. إذا وصلت لأحجام كبيرة أو احتجت صلاحيات دقيقة جداً، سجل تدقيق محاسبي صارم، تطبيق موبايل حقيقي، أو ربط دفع إلكتروني، ستحتاج قاعدة بيانات حقيقية لاحقاً. لكن هذه النسخة تكفي لتشغيل شركة عمرة صغيرة/متوسطة كبداية عملية.
